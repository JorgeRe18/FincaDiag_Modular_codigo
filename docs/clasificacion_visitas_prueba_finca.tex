\subsection{Clasificacion funcional de las visitas}

La clasificacion de las visitas del arbol \texttt{Prueba\_Finca} no se plantea
en terminos de ``visitas buenas'' o ``malas'', sino en funcion de su utilidad
para el proyecto. En la practica, las visitas pueden agruparse en cuatro
conjuntos: fase exploratoria del canal serial, fase de transicion con mezcla
parcial de artefactos, fase madura de captura con PCAP, y visitas de referencia
o control.

\subsubsection{Lectura sintetica}

\begin{itemize}
\item Febrero e inicios de marzo corresponden a una fase exploratoria del canal serial.
\item Mitad de marzo corresponde a una fase de transicion con mezcla parcial de artefactos.
\item Desde el 20 de marzo de 2026 se observa una fase madura de captura con mayor valor para analisis de red.
\item La mejor visita patron es \texttt{Visita\_01\_04\_2026}.
\item La mejor visita para afinar eventos de vaca es \texttt{Visita\_03\_04\_2026}.
\item La mejor visita para variantes del protocolo serial es \texttt{Visita\_02\_04\_2026}.
\item Las mejores visitas para anomalias mixtas de red y operacion son \texttt{Visita\_31\_03\_2026} y \texttt{Visita\_04\_04\_2026}.
\end{itemize}

\subsubsection{Recomendacion de uso}

\begin{enumerate}
\item Usar \texttt{Visita\_03\_04\_2026} y \texttt{Visita\_02\_04\_2026} para afinar el parser y la maquina de estados de vaca.
\item Usar \texttt{Visita\_01\_04\_2026} como ejemplo de visita regular o patron.
\item Usar \texttt{Visita\_31\_03\_2026} y \texttt{Visita\_04\_04\_2026} para documentar degradacion de red y su impacto analitico.
\item Usar el tramo \texttt{20/03/2026 -- 30/03/2026} para comparacion longitudinal y consolidado global.
\end{enumerate}
